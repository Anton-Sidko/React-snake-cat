# Snake cat

### This game create using React and TypeScript

Link to [preview](https://snake-cat.netlify.app/) on Netlify.

### Game features:

- You can adjust size of game field and game speed.
- You can choose field with boundaries or not.
- When game has finished you can play again with same setting or restart game with settings by default.
- State management in this project leverage React Context.
- Styling app with Styled Components and in some places SCSS.
